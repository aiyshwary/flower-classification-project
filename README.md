# Flower Classification using CNNs

Classifies five flower types — **Daisy, Dandelion, Rose, Sunflower, Tulip** — using a custom CNN built with TensorFlow/Keras.

## Dataset
Download the dataset from [Kaggle - Flowers Recognition](https://www.kaggle.com/datasets/alxmamaev/flowers-recognition) and place it as:
```
flowers/
├── Daisy/
├── Dandelion/
├── Rose/
├── Sunflower/
└── Tulip/
```
> The `flowers/` folder and `data.pickle` are excluded from this repo via `.gitignore` due to size.

## Setup
```bash
pip install tensorflow opencv-python scikit-learn matplotlib seaborn
```

## Usage
Run all cells in `Flower Classification project.ipynb` top to bottom.
